			function stopwatch() {
			    static $start = null;
			    if (empty($start)) {
			        $start = microtime(true);
			    } else {
			        $end = microtime(true);
			        $execution_time = $end - $start;
			        $start = null;
			        return $execution_time;
			    }
			}

Модификаторы доступа

Применяется к свойствам и методам

  • public - доступно отовсюду
  • protected - доступен внутри класса и в наследниках
  • private - доступен только внутри класса

Пример

			class Test {
				public $var1 = "test";
				protected $var2 = "test2";
				private $var3 = "test3";
			}
			$test = new Test;
			echo $test->var1;
			echo $test->var2; Выдаст ошибку 
			echo $test->var3; Выдаст ошибку

Пример

			class Test {
				public $var1 = "test";
				protected $var2 = "test2";
				private $var3 = "test3";
				function some(){
					echo $this->var1;
					echo $this->var2;
					echo $this->var3;
				}
			}

Вендорные префиксы

Это приставки (префиксы), используемые производителями (вендорами) браузеров для экспериментальных, еще не принятых в стандарт, CSS-свойств. Приведу список некоторых префиксов для наиболее распространенных браузеров.

Вендорные префиксы

  • -o- — префикс для браузера Опера
  • -moz- — префикс для браузеров из семейства Mozilla
  • -ms — префикс для Интернет Експлорера 8
  • -webkit- — префикс для браузеров, построенных на движке Webkit
  • -icab- — префикс для браузера iCab 
		
Правильное использование

		div {
			-webkit-box-shadow: 0 0 10px #000;	
			-moz-box-shadow: 0 0 10px #000;	
			box-shadow: 0 0 10px #000;		
		}
