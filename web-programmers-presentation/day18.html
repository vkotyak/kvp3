DISTINCT убираем дубли

Выведем все страны из таблицы покупатели

Выведем уникальные страны из таблицы покупатели

HAVING - как WHERE для группировок

		SELECT column_name(s)
		FROM table_name
		WHERE condition
		GROUP BY column_name(s)
		HAVING condition
		ORDER BY column_name(s);

Пример

		SELECT COUNT(CustomerID), Country
		FROM Customers
		GROUP BY Country
		HAVING COUNT(CustomerID) > 5
		ORDER BY COUNT(CustomerID) DESC;
