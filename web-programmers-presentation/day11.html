Использование статического вызова ::

			class A {
				public function a(){
					echo "test";
				}
			}
			A::a(); //можно вызвать так, но нельзя
			$a = new A(); //нужно так
			$a->a();

Статический вызов для статических методов

			class A {
				static public function a(){
					echo "test";
				}
			}
			A::a(); //теперь можно

Классы как типы данных

Типы данных определяют, что можно делать с переменными этого типа. Классы по сути грают ту же роль. Класс объекта решает, что вы можете с ним сделать и какие методы и свойства класса использовать.

Работа с классами

			class ParentClass{}			 
			class MyClass extends ParentClass{}			 
			$a = new MyClass;			 
			var_dump($a instanceof MyClass); // вернет true
			var_dump($a instanceof ParentClass); // вернет true

Проверка типов

			function test(array $a) {
				var_dump($a);
			}

Так же работает и для классов

			function test(MyClass $a) {
				var_dump($a);
			}
