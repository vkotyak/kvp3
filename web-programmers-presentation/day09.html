Функции высшего порядка

Могут принимать другие функции как аргументы.

Могуг возвращаться другие функции как аргументы.

PHP позволяет писать программы в функциональной парадигме, но это совсем другая история.

Применение функций высшего порядка

function filter_array($arr, $filterFunct){
	$result = array();
	foreach($arr as $el) {
		if ($filterFunct($el)) $result[] = $el;
	}
	return $result;
}

Применение функций высшего порядка

		$filter1 = function($val){ return $val == 1;};
		var_dump(filter_array($arr, $filter1));
		$filter2 = function($val){ return $val == 2;};
		var_dump(filter_array($arr, $filter2));
		$filter3 = function($val){return $val != "."};
		var_dump(filter_array($arr, $filter3));

ООП

Программирование ориентированное на объекты

ООП - это не только программирование, но и проектирование

Объект - сущность наделенная состоянием и поведением.

Я - это объект. Моё поведение - это то, что я могу делать.

Моё состояние - это все мои характеристики, к примеру возраст или пол

Методы и свойства

Свойста(атрибуты, поля) - это переменные или константы, которыми обладает объект. Мои свойства - возраст, пол, семейное положение.

Методы - это функции объекта, то что объект может делать.

Кот Барсик - это объект. Его свойства или атрибуты - цвет и возраст. Его методы - есть, пить, мяукать.

Поток документа

Порядок отображения элементов на странице называется потоком документа. Блочные элементы отображаются как прямоугольные области, идущие друг за другом сверху вниз, а строчные элементы располагаются сверху вниз и слева направо и при необходимости переносятся на новую строку.

Свойство position

  • static - по умолчанию
  • relative - элемент можно сдвигать относительно родителя. Занимаемое им место сохраняется в потоке. Сдвиг происходит при помощи свойств top, right, bottom, left
  • absolute - элемент вынимается из потока и позиционируются от блока контейнера со свойством не static
  • fixed - фиксирует местоположение относительно окна браузера, удаляется из потока

float

Определяет, по какой стороне будет выравниваться элемент, при этом остальные элементы будут обтекать его с других сторон. Когда значение свойства float равно none, элемент выводится на странице как обычно, при этом допускается, что одна строка обтекающего текста может быть на той же линии, что и сам элемент.

Пример

  • padding: 30px;
  • background-color: gold;
  • margin-right: 10px;
