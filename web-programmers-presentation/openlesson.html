Программирование

Программи́рование — процесс создания компьютерных программ.

По известному выражению Никлауса Вирта «Программы = алгоритмы + структуры данных»; иными словами, ключевыми непосредственными задачами программирования являются создание и использование алгоритмов и структур данных.

Алгоритмы

Алгоритм - это последовательность действий

Давайте придумаем несколько алгоритмов.

А теперь давайте придумаем алгоритм с ветвлением(Если)

Пример

Программируем

  • Гуглим скретч(scratch)
  • Пишем программу: Кот произносит привет 5 раз

Программы на скретче - тоже программы

На взрослых языках программирования вы найдете все тоже самое. Только нужно написать текстом используя синтаксис языка.

Синтаксис - это правила, такие же как грамматические. Вы должны использовать определеный набор слов и символов для того, чтобы программы работали.

Кирпичики из которых строят программы

  • Условия
  • Циклы
  • Переменные(Числа, Строки, Списки)
  • Операторы
  • Функции

Как строить программы?

Программист слова заказчика переводит на язык программирования.

Заказ, Товар, Заявка - превращаются в числа, строки и списки

К примеру: Товар - это строка с названием и число, которое отображает цену товара.

Программист пишет инструкцию компьютеру по манипулированию простыми данными.

HTTP - протокол

HTTP(Hyper Text Transfer Protocol) — широко распространённый протокол передачи данных, изначально предназначенный для передачи гипертекстовых документов (то есть документов, которые могут содержать ссылки, позволяющие организовать переход к другим документам).
