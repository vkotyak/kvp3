Немного о проектировании

Класс - это имя существительное(Пользователь, Правило, Кубик, Страница).

Метод - это глагол, действие(сохранить, получить, отдать, передать, сделать).

Абстрактные классы

  • Из такого класса нельзя создать объект.
  • Класс содержит реализацию не всех методов.
  • От такого класса нужно насследовать потомков
  • Из классов потомков можно создать объекты
  • Применение: обяжет программистов реализовать методы в потомках

Пример абстрактоного класса

		abstract class Page {
			public function render(){
				$this->_renderHeader();
				echo $this->getContent();
				$this->_rendeFooter();
			}
			abstract public function getContent();

Пример реализации

		class IndexPage extends Page {
			public function getContent(){ 
				return "content";
			}
		}
		$page = new Page(); //Произойдет ошибка
		$indexPage = new IndexPage(); //Можно работать с объектом

Интерфейсы - абсолютные абстракции

В php можно объявить абстракции совсем без реализации 

		interface Page {
			public function render();
			public function getContent();
			protected function renderFooter();
		}

Интерфейсы имплементируются, а не наследуются

			class IndexPage implements Page {
				//Тут обязательно реализовуются все методы
			}
